# Indian Web Series Recommendation Dashboard

This project is a Streamlit dashboard that analyzes user ratings of Indian web series and recommends series to users based on similarity.

## Features

* Dataset preview
* Average rating visualization
* Top rated web series
* User-based recommendation system
* Interactive Streamlit dashboard

## Tools and Technologies

* Python
* Streamlit
* Pandas
* Matplotlib
* Scikit-learn

## Project Structure

```
app.py
indian_webseries_ratings.csv
requirements.txt
README.md
```

## How to Run Locally

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the application:

```
streamlit run app.py
```

3. Open the browser and the dashboard will appear.
